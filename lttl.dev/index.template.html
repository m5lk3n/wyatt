Wyatt in its initial version uses the Google Maps API, e.g., for Geocoding to convert geographic coordinates into human-readable addresses. Using Google APIs is NOT free of charge! Please read Platform Pricing & API Costs - Google Maps Platform for details.

Privacy Policy

Wyatt itself does not collect any personal data. The application does not store any data on a server. The user's (API) key is stored securely on the client device along with the user's reminder data (that's the data entered by the user through the application's user interface). The application itself does not use cookies or any other tracking technologies.

Please note that, through the usage of Google Maps, Google Privacy Policy and Google Controller-Controller Data Protection Terms apply.

Permissions

Here's the list of application permissions and for what they are required:

  • Internet: Required to make HTTPS calls to Google APIs.
  • Location: Required to know the user's current location to be able to notify.
  • Notification: Required to be allowed to remind the user.
  • Storage (?): Required to store the user's to-do list data on the client.
